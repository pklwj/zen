# 蓝莓种植攻略

- 蓝莓 L25 是优质的南高丛蓝莓品种，因其**需冷量低、单果大、硬度高、口感脆甜且抗病性强**，非常适合在江浙沪、两广、西南等南方地区大田种植，同时也是阳台、庭院盆栽的明星品种。


- 蓝莓 F6（全称常为 FL05-290，是极具代表性的南高丛脆甜品种）与 L25 一样，同属于近年来风靡一时的大果、高硬度、高糖低酸、常绿型南高丛蓝莓。由于两者的品系和生长习性高度相似，F6 的种植大体上可以参考 L25 的方法，但 F6 在长势、修剪和水肥缓冲上有其自身的独特个性。

# 1、基础
## 盆
如果是苔藓苗，推荐用1.5加仑-2加仑的普通加仑盆育苗，如果是钵苗，推荐直接上5加仑矮胖盆；

## 土
- 首要要求：是土壤的PH值，PH值要介于4.3-5.5之间，最适宜的在4.5左右，高于5.5之后蓝莓会开始出现不同程度的发育障碍，高PH值对蓝莓的主要功能性伤害，体现在蓝莓对铁元素的吸收上出现问题，直观的表现是出现大量的嫩叶发黄现象（嫩叶脉间失绿）

- 次要要求：是土壤必须疏松透气，蓝莓没有大的主根和粗壮的根枝，基本上以细嫩的须根系为主，细嫩的须根系对土壤积水的耐受度很低，因为细小，所以细菌和病菌侵入后，根系细胞防御纵深很小，浆化板结的土壤容易积水泥泞，使得根系长期泡在水中引发腐烂，同时不疏松透气的环境，使得根系周围的土壤益生菌的生存环境恶劣，大量有害厌氧菌增殖，恶化土壤生态环境，增加易腐几率
  
- 最后要求：是必须富含腐殖质，蓝莓嫌钙和嫌氯

- 土配比：松针腐殖土 6 份 + 发酵松鳞 (6~12mm) 2 份 + 大颗粒珍珠岩 (3~5mm) 1.5 份 + 发酵羊粪粉 0.5 份

## 肥料
一些必须的肥料：**硫酸钾复合肥**（史丹利硫基硫酸钾复合肥），**矿源黄腐酸钾**，**羊粪蛋儿**等，一般有这三种差不多基本可以了；

### 购买材料：
腐殖土、发酵松鳞（颗粒度 6~12mm）、大颗粒珍珠岩（颗粒度 3~5mm）、陶粒（直径 1~2 厘米）
羊粪
PH试纸

# 2、种植管理
## 浇水
夏季需要每日浇水，春秋可以2-3天浇水一次，冬季未零下时一周1次，零下后不浇水。

## 浇肥
小苗以长枝叶为主，所以氮磷钾在均衡的基础上以氮肥为重，化肥可以用硫基硫酸钾复合肥、硫酸铵等，有机肥以腐熟的饼肥和羊粪为主，到了10月份（杭州），就要有重点的加大磷钾肥的使用，磷酸二氢钾可以每隔2周叶面喷施一次，等蓝莓树完全落叶或者红叶后，就可以起盆修根换土加底肥，底肥以腐熟饼肥和羊粪等为好，开春花未展开前，加施磷酸二氢钾叶面喷施一次，花蕾出现后停止施肥，挂果后要进行果期追肥，用腐熟各类粪进行土表面浅埋追肥，采果后一般为暑期，停止施肥，等9月份左右气温下降至30度内进行第二生长期追肥。

## 修剪
蓝莓修剪分春抹、秋抹、夏剪、冬剪。
春季和秋季都是生长季，也是抽枝萌芽的季节，所以采用节约植株损耗的抹芽的方法，抹去内膛方向芽、过密的芽、徒长枝的顶芽、细弱枝的芽等等一切以后发展方向不好的芽，让全株营养集中到几个希望继续生长的芽点上；

夏季修剪是在果后，果后立刻修剪老枝、已结果枝，促使新的枝条萌发出来，夏剪最好在8月底前完成，最好是果后一周内修剪，修剪的太晚，新萌发的枝条不能及时完成花芽分化影响来年产量。

冬季修剪一般在落叶或红叶后，此时花芽展露无疑，可以根据树体的强盛，选择留多少花芽，多余的花芽、羸弱的花芽需要剪除，避免来年耗费营养；

除了四个季节的目的性很强的修剪外，闲暇时刻的修枝也是一件让人赏心悦目和乐此不彼的事情，修枝的主要目的有两个：控制树型和增强光照，所以修剪的时候，要根据每棵树的情况，剪除多余的伏地枝、内膛枝、细弱枝、病枝等，同时对一些生长角度怪异的枝条进行拉枝造型，寻找最美观和最有光照的角度，枝条健壮后，离土面15厘米以下叶片摘除掉；
