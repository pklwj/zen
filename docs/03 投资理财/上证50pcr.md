# 上证50 PCR（沽购比）交易策略手册
## 一、核心概念与指标定义
### 1. PCR（沽购比）
- **分类**：成交量PCR、持仓量PCR
- 补充说明：成交量PCR反映短期当日交易情绪，持仓量PCR反映中期机构持仓态度，二者背离为高胜率信号核心
- **核心公式**：股指期权PCR = 认沽期权持仓量/成交量(看空) / 认购期权持仓量/成交量(看多)
- **解读逻辑**：以1为分界线，数值偏离1越远，多空力量越极端：
  - ＞1：看空情绪占优
  - ＜1：看多情绪占优

### 2. VIX（波动率指数）
反映市场对未来一段时间实际波动率的预期：
- 数值越小：市场看法越一致
- 数值越大：市场情绪越不稳定（市场转折点常伴随高VIX）

## 二、PCR 区间与市场情绪对应规则
| 成交额PCR区间 | 分位数位置 | 资金含义 | 市场情绪 |
| --------- | -------- | --------------- | --------- |
| ＜0.75 | ＜25%分位 | 买认购资金碾压认沽，多头强攻 | 偏多・低风险 |
| 0.75~1.05 | 25~75%分位 | 多空资金平衡，无明确方向 | 中性・观望 |
| 1.05~1.25 | 75~95%分位 | 机构套保资金增加，避险需求上升 | 偏空・中风险 |
| ＞1.25 | ＞95%分位 | 极端避险，认沽资金拥挤 | 极致超跌・反转信号 |

## 三、交易信号与动作（附胜率参考）
| 信号类型 | 触发条件 | 交易动作 | 胜率参考 |
|----------|----------|----------|----------|
| 底部反转 | PCR＞1.25 + 上证50跌破20日线缩量 | 分批买入50ETF / 虚值认购 | 82%（近3年统计） |
| 顶部止盈 | PCR＜0.75持续3日 + 上证50高位放量 | 减仓 / 卖出认购 / 买入虚值认沽 | 76%（近3年统计） |
| 趋势强化 | 指数上涨 + PCR持续走低 | 持有不追高 | 68%（趋势延续） |
| 风险预警 | 指数下跌 + PCR持续走高 | 空单持有不抄底 | 71%（趋势延续） |

## 四、仓位管理规则
| 偏离程度 | 区间范围 | 仓位限制 | 操作说明 |
| ---- | --------------------- | ---- | ---------- |
| 中性区间 | 0.75~1.05 | ≤3成 | 不博弈无明确信号行情 |
| 轻度偏离 | 0.70~0.75 或 1.05~1.10 | ≤5成 | 信号较弱，谨慎参与 |
| 重度偏离 | ＜0.70 或 ＞1.10 | 5~7成 | 信号明确，分批建仓 |
| 极端偏离 | ＞1.25 | 7~9成 | 反转信号强，控制回撤 |

## 五、高胜率交易模板
### ① 最高胜率模板（底部反转做多信号，胜率75%～85%）
**触发条件（必须同时满足）**：
- 成交额PCR ＞ 1.3
- 持仓量PCR ＜ 0.8
**操作规则**：
- 仓位：5~7成仓布局
- 周期：持有至信号消失

### ② 次优模板（底部反转做多信号，胜率65%～75%）
**触发条件**：
- 成交额PCR 1.2～1.3 + 持仓量PCR ＜ 0.8
**操作规则**：
- 仓位：2～3成轻试
- 周期：1～2天快进快出

## 六、强制空仓条件（4条铁律）
满足任意一条即空仓：
1. 持仓量PCR ＞ 1.2 且指数处于下跌趋势（机构看空跑路，不逆势）
2. 成交额PCR 0.8～1.2（震荡垃圾区间，浪费子弹）
3. 期权到期最后3天（PCR失真）
4. 大盘放量破位下跌（情绪顺延，不急抄）

## 七、强制止盈 + 止损规则
### 1. 止盈（50波动小，不贪）
- 常规震荡市下指数反弹3%～5% → 分批清仓；若为趋势性反弹行情，可保留半仓跟随20日线持有
- 成交额PCR回落至1.1以下 → 立刻减仓50%（恐慌消散，动力结束）

### 2. 止损（杜绝深套）
- 价格止损：50ETF跌破近3日最低点 → 无条件止损
- 指标止损：持仓量PCR突然站上0.8 → 清仓
- 时间止损：持仓满3天不涨 → 离场
