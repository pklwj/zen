# 选股策略
高股息
低PB
避免估值陷阱

### 核心筛选量化模型
多因子初选模型：

| **维度**   | **指标门槛（参考）** | **逻辑说明**           |
| -------- | ------------ | ------------------ |
| **回报率**  | 股息率 > 5%     | 显著高于无风险收益率（如国债收益率） |
| **盈利能力** | ROE > 10%    | 确保公司具备基本的赚钱效率      |
| **估值水平** | PB < 1.5倍    | 确保资产价格未被过度炒作       |
| **财务健康** | 经营性现金流为正     | 确保分红有现金支撑          |
| **成长潜力** | 营收增长率 > 0    | 哪怕是慢节奏，也不能是衰退型公司   |
