**综合收益率 = (1 - 分红支付率)×ROE + 股息率**
# 行业基准筛选（排除 "高股息陷阱"）
- **高股息行业**（银行 / 公用事业 / 传统消费）：综合收益率≥8%
- **中等股息行业**（制造业 / 医药）：综合收益率≥10%
- **成长行业**（科技 / 新能源）：综合收益率≥12%

# 分红质量三重验证（确保收益可持续）
- 分红连续性：连续 5 年以上稳定分红（每年分红≥前一年 80%）
- 分红比例：分红支付率稳定在 30%-70% 区间（太低保守，太高透支发展）
- 分红增长：近 3 年股息复合增长率≥0（最好 > 5%）

# 盈利能力与质量（确保 "真成长"）
- ROE 门槛：连续 3 年 ROE≥12%（优质企业通常 > 15%）
- ROE 稳定性：近 3 年 ROE 波动≤20%（避免大起大落）
- 盈利真实性：经营现金流 / 净利润≥0.8（排除 "纸面富贵"）
- 财务安全：资产负债率 < 60%（越低越好，尤其对高股息股）

# 估值匹配度（避免 "价值陷阱"）
- PE 与 ROE 对比：PE/ROE≤1（彼得林奇法则，越低越安全）
- 股息率 vs 国债：股息率≥2 倍 10 年期国债收益率（当前约 5%）
- 历史分位：PE 处于近 5 年历史低位（<30% 分位最佳）
