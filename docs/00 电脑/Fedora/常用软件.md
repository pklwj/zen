# sway

## 公用程式
sway lock
簡單的螢幕鎖。

waybar
針對基於 Sway 和 Wlroots 的合成器的可高度自訂的 Wayland 欄位。

rofi
視窗切換器、應用程式啟動器和 dmenu 替換程式。

dunst
高度可設定且輕量級的通知常駐程式。

kanshi
動態顯示器組態工具。

Thunar
適用於 Linux 和其他類 Unix 系統的檔案管理器。

## 組態設定

foot
Foot 是一款快速、輕量、簡約的 Wayland 終端模擬器。


brightnessctl
brightnessctl allows users to read and control device brightness.


imv
imv 是一款命令列影像檢視器，專為平鋪式視窗管理器設計。
