#怎么选
#### ​**​常见肥瘦比例及特点​**​
| ​**​比例​**​  | ​**​口感特点​**​       | ​**​适用场景​**​ | ​**​推荐人群​**​  |
| ----------- | ------------------ | ------------ | ------------- |
| ​**​2:8​**​ | 极瘦肉感，紧实微甜，酒香突出     | 家庭烹饪、煲仔饭、炒菜  | 健身爱好者、低脂需求者   |
| ​**​3:7​**​ | 经典广式风味，肥肉融油不腻，瘦肉鲜嫩 | 煲仔饭、蒸煮、即食    | 大众消费者、传统风味爱好者 |
| ​**​4:6​**​ | 油脂更丰富，香气浓郁，口感软糯    | 炒菜、炒饭、下酒菜    | 偏好醇厚口感的中老年群体  |
| ​**​5:5​**​ | 肥瘦均衡，入口即化，略带嚼劲     | 即食、凉拌、配粥     | 儿童、牙口较弱者      |
