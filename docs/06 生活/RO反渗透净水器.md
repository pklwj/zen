#  额定净水量
在最新国标《净水机水效限定值及水效等级》（GB 34914-2021）中，将额定总净水量分为3个等级：
- 第一级水效标准：额定总净水量≥4000升
- 第二级水效标准：额定总净水量≥3000升
- 第三级水效标准：额定总净水量≥2000升

# 看净水率（废水比）
根据最新国标《净水机水效限定值及水效等级》（GB 34914-2021）定义的标准中，净水产水率分为3级：
- 第一级水效净水产水率≥65%
- 第二级水效净水产水率≥55%
- 第三级水效净水产水率≥45%

# 看膜规格/通量
流量的计算方式是膜规格除以24小时除以60分钟，等于每分钟能过滤多少L（升）水；膜规格一般用单位G（加仑）展示。**1G=3.785L**
